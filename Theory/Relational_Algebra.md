# Relational Algebra

It is a procedural query language and a set of operations on relations.

## Fundamental Operations 

- **Select** `\sigma`
- **Project** `\prod`
- **Union** `\bigcup`
- **Set difference** `-`
- **Cartesion Product** `\times`
- **Rename** `\ro`

## Additional Operations 

- **Intersection** `\bigcap`
- **Assignment** `=`
- **Natural Join** `\varphi`
- **Division** `/`
- **Outher join**: left, right and full
